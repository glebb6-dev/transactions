# Credit Card Churn Analysis

## Описание проекта
Проект посвящён анализу оттока клиентов банка на основе данных о владельцах кредитных карт.
Цель проекта — изучить характеристики клиентов и выявить факторы, связанные с уходом клиентов.

---

## Данные
Используется открытый датасет Credit Card Customers (BankChurners.csv),
содержащий информацию о более чем 10 000 клиентах банка.

---

## Что сделано в проекте
- загрузка и первичный анализ данных;
- проверка данных на пропуски;
- фильтрация и сортировка данных;
- создание новых признаков;
- группировка и агрегация данных;
- визуализация результатов.

---

## Используемые технологии
- Python
- pandas
- Jupyter Notebook
